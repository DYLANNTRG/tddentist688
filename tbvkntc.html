    Kích hoạt Menu trợ năng

    Bạn có thể bật menu trợ năng bằng cách nhấp vào biểu tượng menu trợ năng xuất hiện ở góc trang. Sau khi kích hoạt menu hỗ trợ, vui lòng đợi một lát để menu hỗ trợ tải toàn bộ.


    Sáng kiến khả năng truy cập web

    Nếu bạn gặp khó khăn khi xem các trang web, Sáng kiến Truy cập Web sẽ cung cấp các mẹo toàn diện để tùy chỉnh trình duyệt web và thiết lập máy tính nhằm cải thiện trải nghiệm trực tuyến của bạn:


  • Duyệt web bằng bàn phím

  • Phóng to văn bản và hình ảnh

  • Thay đổi màu sắc và phông chữ

  • Tùy chỉnh con trỏ chuột


    • Phương thức nhập thay thế

    Nếu bạn cần các lựa chọn thay thế cho phương thức nhập liệu truyền thống như chuột và bàn phím thì có một số ứng dụng nhận dạng giọng nói có thể giúp bạn điều hướng trên web. Dragon Natural Nói là một trong những lựa chọn như vậy. Phần mềm này có thể hỗ trợ điều hướng một trang web bằng điều khiển bằng giọng nói.
